# External Benchmarks

Reference sites worth learning from. Reviewed 2026-03-18.

## Gray Zone Warfare
- https://www.grayzonewarfare.com/about
- https://www.grayzonewarfare.com/news/roadmap
- Pillar-based about page, concrete survival/combat/world claims
- Roadmap split into near-term named updates and longer-term horizon

## SCUM
- https://scumgame.com/index.php/about-page/
- System-first public pitch (metabolism, crafting, loot, base building as distinct pillars)

## Star Citizen
- https://robertsspaceindustries.com/en/development
- Public roadmap, public telemetry, issue reporting, patch notes, visible developer activity

## DayZ
- https://forums.dayz.com/topic/237979-faq/
- Official status reports, bug tracker, community server stance, modding/server files surfaced

## Mullvad
- https://mullvad.net/en/why-mullvad-vpn
- "What sets us apart" page with explicit trust claims backed by mechanisms (open source, audits, GPG key)

## Tails
- https://tails.net/install/download/
- Download + mirror links + verification integrated into one user flow
