ALIS

Learn. Survive. Connect.

Open-source post-apocalyptic survival game built on Unreal Engine 5. Real places as the world. Systems grounded in real-world logic. Open code and a pact for honest design.

Systems Status Vision Trust

What's Built

  • Survival physiology — metabolism, thresholds, cascading states
  • Server-authoritative inventory — grid packing, weight, equipment storage
  • Mind — inner voice driven by state and surroundings
  • World pipeline — real terrain, real building footprints, real elevation
  • Data architecture — JSON-driven tuning, built for development with AI agents
  • Descriptor-driven UI, dialogue, and interaction frameworks
  • Signed build and release chain

In Progress

  • First world tile — 8 km × 8 km dense urban zone
  • Multiplayer infrastructure and dedicated servers
  • Launcher — updates, verification, secure game handoff
  • Deeper survival loops and crafting

Trust

  • Full source code under AGPL-3.0
  • Signed releases with public verification key
  • Architecture and systems documentation
  • The Alis Pact — commitments on fair play and licensing
